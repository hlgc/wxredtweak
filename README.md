#iOS版微信抢红包插件
##安装方法
使用Theos安装
##注意
运行源码时把Makefile中的 THEOS_DEVICE_IP = 192.168.100.138 IP地址需要换成自己越狱手机中的IP地址,本插件适用于微信7.0.0的版本
